# termux-box64

# English | [한국어](README_KO.md)

Install Box64 in termux natively without wine.

This script installs box64 without wine setup.

To get started,

```bash
curl -LO https://raw.githubusercontent.com/hajs110208/termux-box64/refs/heads/main/install.sh && chmod +x install.sh && ./install.sh
```

To run Box64,

```bash
box64 [program]
```
